# Stock-Supply-Power-BI-Dashboard

## Project Overview

This project analyzes supply chain operations including **sales, inventory, warehouses, and supplier performance** using Power BI.

The dashboard helps businesses monitor stock levels, supplier reliability, and purchasing activity.

## Dataset
The dataset includes:

* 100,000+ Sales transactions
* 150,000+ Inventory records
* 50,000+ Purchase records
* 200 Products
* 10 Warehouses
* 50 Suppliers

## Dashboard Pages

### 1. Executive Overview
Key KPIs:

* Total Sales Value
* Inventory Value
* Inventory Turnover
* Fill Rate %

### 2. Sales Analysis
* Product sales performance
* Monthly sales trend
* Top 10 products by revenue

### 3. Inventory Analysis
* Warehouse stock distribution
* Inventory value by category
* Stock trend over time

### 4. Supplier Performance
* Supplier purchase value
* Ordered vs received quantity
* Fill rate analysis
